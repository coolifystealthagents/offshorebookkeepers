# Design summary

- Design concept: Bookkeeper ledger
- Layout personality: narrow editorial ledger
- Typography: Cormorant Garamond headings with IBM Plex Sans body text
- Color palette: primary #064e3b, secondary #6ee7b7, accent #b45309, background #ecfdf5, text #06281f, muted #64748b
- Shape language: Slight rounding
- Visual motif: ledger lines and receipt panels
- Unique layout choices: first sections use Financial ledger, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
